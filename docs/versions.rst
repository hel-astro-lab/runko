Versions
========

Runko uses a ``major.minor`` versioning scheme for its official releases, e.g. ``1.0``, ``2.1``.
Major versions typically introduce new features, major code refactoring, and other large changes.
Minor versions contain smaller bug fixes and enhancements.

See the `GitHub release history <https://github.com/hel-astro-lab/runko/releases>`_ for full details and a changelog.

The main development branch is ``master``.
