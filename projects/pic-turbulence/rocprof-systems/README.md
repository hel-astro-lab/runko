Install rocprofiler-systems like:

mkdir install
cd install
wget https://github.com/ROCm/rocprofiler-systems/releases/download/rocm-6.4.4/rocprofiler-systems-1.0.2-opensuse-15.6-PAPI-OMPT-Python3.sh
./rocprofiler-systems-1.0.2-opensuse-15.6-PAPI-OMPT-Python3.sh

And select no. Then after setting the paths as is done in the launch script, the config can be generated by:
srun -p small-g --gpus 1 rocprof-sys-avail -G rocprof-sys.cfg
